## 概要
星空の写真に特化した画像加工を行う、Android向けの個人開発アプリです。
天体写真をスマートフォン上で手軽に調整できることを目的として制作しました。

##制作背景
既存の画像加工アプリを使用する中で、ツールの数が多く操作が複雑であることや、
加工手順を別途調べながら作業する必要がある点に不便さを感じました。
ユーザーは写真を選択するだけで、あらかじめ設定された加工手順に沿って編集し、画面に表示する画像加工のパラメータをを一つにすることによって、ツールを探す手間や加工手順を調べる手間を省けるものを作りました。

## 今後の課題
・端末や画面サイズの違いを十分に考慮したレイアウト設計まで手が回らなかった
・ Undo機能など、操作履歴を考慮した設計を実装できなかった
・ 画像処理の負荷と画質のバランス調整に課題が残った

##画像を編集する手順
①明るさ（少し明るくして、暗い星の光を見えるようにしよう）
②コントラスト（写真の色の明るさの差を上げて星の輝きにメリハリをつけよう）
③シャドウ　背景が暗いなら不自然にならないように暗いところを明るくしよう
④色相　　空の色が赤や紫がかっていたら反対色を強くしてバランスを整えよう。青を強くすると宇宙観が出るよ
⑤彩度　不自然にならないように彩度を上げよう、この作業はしなくてもいいよ
⑥ぼやかし　写真をぼやかして星の光をふんわりさせたり、よりシャープにして星の光をはっきりさせることもできるよ
